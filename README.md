# 🛒 React eCommerce Platform

A full-featured eCommerce web application built with React, supporting product browsing, cart management, order processing, and real-time order tracking with backend integration.

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/DiceTech435/react-ecommerce.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```
